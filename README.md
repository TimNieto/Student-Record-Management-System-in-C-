# Student Record Management System in C++

A simple C++ console application for managing student records. This program allows you to add, search, display, and delete student information in an organized manner.

## Features

- Add new student records including:
  - Student ID
  - Full Name
  - Birthday
  - Address
  - Gender
  - Degree Program
  - Year Level
- Search for a student by ID or full name.
- Display all student records.
- Display a specific student record.
- Delete a student record.
- Menu-driven interface for easy navigation.

## Technologies Used

- C++ (Standard Library)
- Vector for storing student records
- Console-based input/output
